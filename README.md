# About

This is a basic project for an API server with basic JWT token-based authentication from
a configuration file.

To be used as a quick-starter for other API based projects.

# Usage

### Start server
```
$ npm start
```

### Run tests
```
$ npm run tests
$ npm run test-coverage
```
